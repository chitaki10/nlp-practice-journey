# 🧠 NLP Practice Journey

This repository documents my hands-on journey of learning **Natural Language Processing (NLP)** from fundamentals to practical implementations using Python.

The goal of this project is to build a **strong intuition of NLP preprocessing techniques**, understand how raw text becomes structured data, and implement these concepts step-by-step using popular libraries like **NLTK and Python**.

---

# 📚 Topics Covered

This repository focuses on the **core preprocessing stages of NLP**, which form the foundation for many advanced applications such as chatbots, search engines, recommendation systems, and large language models.

The following concepts are implemented:

* Text Tokenization
* Stemming
* Lemmatization
* Stopword Removal
* Part-of-Speech (POS) Tagging
* Named Entity Recognition (NER)

Each concept is implemented in **Jupyter Notebooks with explanations and code examples**.

### Notebook Descriptions

| Notebook                        | Description                                             |
| ------------------------------- | ------------------------------------------------------- |
| **01 Tokenization**             | Splitting raw text into words or sentences              |
| **02 Stemming**                 | Reducing words to their root form                       |
| **03 Lemmatization**            | Converting words to their meaningful base form          |
| **04 Stopwords Removal**        | Removing common words that add little meaning           |
| **05 POS Tagging**              | Identifying grammatical roles of words                  |
| **06 Named Entity Recognition** | Detecting entities like names, locations, organizations |

# ⚙️ Installation

Clone this repository:

```bash
git clone https://github.com/yourusername/nlp-practice-journey.git
cd nlp-practice-journey
```

Install required dependencies:

```bash
pip install nltk jupyter
```

Download NLTK datasets:

```python
import nltk
nltk.download('all')
```

# 🚀 Running the Notebooks

Start Jupyter Notebook:

